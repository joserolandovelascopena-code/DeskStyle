# DeskStyle Tienda Online

DeskStyle es una plataforma en línea dedicada a la venta de objetos y accesorios decorativos para espacios de trabajo, como escritorios, oficinas y habitaciones del hogar. Su objetivo es ofrecer una amplia variedad de productos decorativos en un solo lugar, permitiendo que los usuarios puedan buscar, seleccionar y comprar los accesorios que necesitan desde su celular o computadora, realizando todo su pedido de forma rápida y sencilla desde un único sitio web.

## Funciones principales

- Registro de usuarios.
- Inicio de sesión para los usuarios del sistema.
- Catálogo de categorías de productos disponibles en el inventario.
- Atención al cliente personalizada.
- Realización de pedidos y confirmación de la compra mediante el envío de un correo electrónico.

## Instalación del proyecto

Para instalar el proyecto de forma local en una computadora con conexión a Internet, ejecuta los siguientes comandos desde PowerShell o Git Bash.

### 1. Prerrequisitos

Asegúrate de tener instalado el siguiente software:

- [Git](https://git-scm.com)

### 2. Clonar el repositorio

Clona este repositorio en tu máquina local utilizando el siguiente comando:

```bash
git clone https://github.com/joserolandovelascopena-code/DeskStyle.git
```
